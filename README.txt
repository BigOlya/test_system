Инструкция по запуску файла app2.py

1. Скачать архив из GitHub по ссылке https://github.com/BigOlya/test_system/tree/master
2. Разархивировать папку на свой ПК
3. Скачать Python с официального сайта https://www.python.org/downloads/
4. Запустить командную строку от имени админимтратора
5. Вставить команду pip install Flask для установки фреймворка Flask
6. После успешной установки ввести команду cd <путь до папки, в которой расположен файл app2.py из шага 2>
7. Ввести команду python app2.py
8. В браузере ввести адрес http://127.0.0.1:5000
